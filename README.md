# mail
依赖：
boost 1.58 
openssl： 由于openssl在win下有直接的二进制安装包， 我是直接下载最新版本进行使用的。


编译平台 ： VS2013

需要设置的系统环境变量：
BOOST_ROOT : 你的boost根目录
BOOST_LIB  : 你的boost编译的库文件目录
OPENSSL_INCLUDE : 你的openssl库目录（如果你下载的是dll， 请勿忘记把dll放到程序运行目录）
